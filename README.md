# PAQ-in-color-vision
This repository contains the code, data, and supplementary materials for the paper "Learning the eye of the beholder:
Statistical modeling and estimation for personalized color perception" by Xuanzhou Chen, Austin Xu, Jingyan Wang and Ashwin Pananjady. The paper was presented at ICML Workshop, 2024.

### Content
The repository includes:
- Python scripts for simulations, statistical tests and user study.
- PAQ response datasets collected by color perception test.
- Supplementary figures, e.g., ellipse and copunctal point plots.
